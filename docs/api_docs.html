Tài liệu API

Hướng dẫn chi tiết về cách sử dụng API

Chỉ có ADMIN mới có quyền sử dụng API, nếu bạn muốn sử dụng vui lòng liên hệ ADMIN để cấp API Key.
API Users
Endpoint

GET /api/Post.php

Các tham số
  • api (Bắt buộc): API key để xác thực.
    Ví dụ: ?api=your_api_key
  • username (Tùy chọn): Lọc theo tên người dùng.
    Ví dụ: &username=johndoe
  • desc (Tùy chọn): Lọc theo mô tả người dùng.
    Ví dụ: &desc=developer
  • sort (Tùy chọn): Sắp xếp kết quả theo cột và thứ tự (asc hoặc desc).
    Ví dụ: &sort=username:asc hoặc &sort=created_at:desc
Ví dụ sử dụng đầy đủ

https://example.com/api/Post.php?api=your_api_key&username=johndoe&desc=developer&sort=username:asc
