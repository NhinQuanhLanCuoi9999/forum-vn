Tài liệu API

Hướng dẫn chi tiết về cách sử dụng API

Chỉ có ADMIN mới có quyền sử dụng API, nếu bạn muốn sử dụng vui lòng liên hệ ADMIN để cấp API Key.
API Users
Endpoint
GET /api/User.php
Xác thực

Dùng header: 

Authorization: Bearer YOUR_API_KEY

Tham số
  • username (Tùy chọn): Lọc theo tên người dùng.
  • desc (Tùy chọn): Lọc theo mô tả.
  • sort (Tùy chọn): sort=username:asc
Ví dụ sử dụng
curl -H "Authorization: Bearer YOUR_API_KEY" "https://example.com/api/User.php?username=johndoe&desc=developer&sort=username:asc"
API Bans
Endpoint
GET /api/Bans.php
Xác thực
Authorization: Bearer YOUR_API_KEY
Tham số
  • username (Tùy chọn): Lọc theo tên người dùng.
  • ip (Tùy chọn): Lọc theo địa chỉ IP.
  • sort (Tùy chọn): sort=ban_start:desc
Ví dụ sử dụng
curl -H "Authorization: Bearer YOUR_API_KEY" "https://example.com/api/Bans.php?username=johndoe&ip=192.168.1.1&sort=ban_start:desc"
